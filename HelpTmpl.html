This Google Apps script exports test coverage data from a salesforce org to the active Spreadsheet. The script is intended to help SFDC developers to get the full state of test coverage of the programming code in current developer org. In order to use this script you need to create a Connection Apps in your Salesforce Org (for details please see hear).

As Callback Url in your Connection Apps setting in your salesforce org please specify https://script.google.com/macros/d/.

For more information please see the file README.TXT
